Tools

Node.js

Node allows for easy installation of modules and hosting data on a server.

Phaser

Phaser is a very powerful 2D game engine framework. It has loads of utilities for loading graphics and handling physics. Most of our client side is built on or heavily incorporates Phaser

Tiled

Tiled provided us with a very easy tilemap creation interface and customization. We used tilemap to generate the background tiles and the locations of all the walls and rocks.

Piskel

We used Piskel to be able to resize some of the sprites in the sprite sheet and for creating seperate sprite files for each. We had to make the seperate files in order to have physics collisions on the sprites seperate from the tilemap. This included the players, fire, and bomb sprites.

Socket.io

Socket.io is a framework that makes websockets very easy to implement. It is used both server-side and client-side to send communications across the network.
